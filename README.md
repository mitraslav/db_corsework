## Описание

Данный проект представляет собой учебную курсовую работу по работе с базами данных и внешними API.

В рамках проекта:
- выполняется получение данных о работодателях и вакансиях с сайта hh.ru с использованием публичного API;
- проектируется и автоматически создаётся база данных PostgreSQL;
- данные о компаниях и их вакансиях сохраняются в базу данных;
- реализован класс `DBManager` для работы с базой данных и выполнения SQL-запросов;
- предоставляется простой текстовый интерфейс для взаимодействия с пользователем.

Проект демонстрирует навыки работы с:
- REST API (`requests`);
- PostgreSQL;
- SQL-запросами (JOIN, AVG, WHERE, LIKE);
- библиотекой `psycopg2`;
- принципами ООП и базовыми принципами SOLID.

## Установка

1. Клонируйте репозиторий:
```
git clone https://github.com/mitraslav/coursework_1
```
2. Установите зависимости:
```
pip install -r requirements.txt
```
3. Создайте файл `.env` в корне проекта и укажите параметры подключения к PostgreSQL:
```
PG_HOST=localhost
PG_PORT=5432
PG_USER=postgres
PG_PASSWORD=postgres
PG_DB=hh_vacancies
```

Файл `.env` не должен добавляться в репозиторий и находится в `.gitignore`.

## Использование

Для запуска проекта выполните команду из корня проекта:
```
python main.py
```

При запуске:
- автоматически создаётся база данных (если она ещё не существует);
- создаются таблицы `companies` и `vacancies`;
- загружаются данные о работодателях и их вакансиях с сайта hh.ru;
- запускается текстовое меню для работы с данными.

В интерфейсе доступны следующие действия:
1. Просмотр списка компаний и количества вакансий у каждой компании;
2. Просмотр списка всех вакансий;
3. Получение средней зарплаты по вакансиям;
4. Просмотр вакансий с зарплатой выше средней;
5. Поиск вакансий по ключевому слову в названии.
